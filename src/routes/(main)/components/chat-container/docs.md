# Chat Container

A component for creating chat interfaces with intelligent auto-scrolling behavior, designed to provide a smooth experience in conversation interfaces.

## Installation

<Tabs items={["CLI", "Manual"]}>
<Tab value="CLI">

### Using CLI

```bash
npx shadcn-svelte@latest add https://sv-animations.vercel.app/r/chat-container.json
```

</Tab>
<Tab value="Manual">

### Manual Installation

Copy the component source files into your project and update the example files with real usage.

</Tab>
</Tabs>

## Usage

```svelte
<script lang="ts">
  import { ChatContainer } from "$lib/components/ai/chat-container";
</script>

<ChatContainer />
```
