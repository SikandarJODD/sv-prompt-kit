# Full Chatbot

A complete AI chatbot setup with a live frontend, local model config, and a server route.

## What this page shows

- The real `FullChatbot` component, not a mock preview.
- The packages required to run the chatbot.
- The `src/lib/config/ai.ts` setup.
- The `src/routes/api/chat/+server.ts` route.
- The component file group under `src/lib/components/primitives/full-chatbot`.

## Setup order

1. Install packages.
2. Configure `src/lib/config/ai.ts`.
3. Add `src/routes/api/chat/+server.ts`.
4. Copy the primitive component files.
5. Match the documented folder structure.
