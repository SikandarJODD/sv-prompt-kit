# Introduction

Svelte Prompt Kit is a collection of customizable, high-quality components built specifically for AI applications. It makes it easy to design beautiful chat experiences, AI agents, autonomous assistants, and more with minimal effort.

## Built for AI interfaces

Built on top of `shadcn-svelte`, Svelte Prompt Kit follows the same design principles and philosophy. Instead of helping you build a general component library, it focuses specifically on creating polished AI interfaces.

## Actively maintained

This project is actively maintained and continuously evolving. We welcome your feedback, suggestions, and contributions as we expand the component collection and improve existing features.

- Use the docs as the primary reference for installation and usage.
- Open issues or contribute improvements as the component library grows.
- Expect the collection to keep evolving around practical AI interface patterns.
